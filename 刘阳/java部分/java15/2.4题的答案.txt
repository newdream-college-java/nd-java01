第一幅图：错误有两处：

	1.父类的public void method(){}方法；和子类的private void method(){}方法本来这里应该是
	想使用方法重写的套路，但是没有注意自己子类的修饰符的作用域小于了父类的作用域，重写条件
	不成立，所以错！
	2.子类的private void method(){}方法和子类的public int method(){}方法，是想使用方法重载	。但是，参数列表相同，导致了使用方法重载的条件不成立，所以错误！
