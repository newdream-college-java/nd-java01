二、用户操作
1.创建用户
create user '用户名'@'地址' identified by '密码';
如：
create user 'yhm'@'localhost' identified by '123';
注意：地址写127.0.0.1 或者localhost 表示本机地址（回环地址），但是如果我们连接的不是本机mysql数据库服务器，那么就要写服务器的ip地址。
2、使用该用户去登陆mysql数据
		mysql -u 用户名 -p
		输入密码
如：mysql -u yhm -p
密码 123
		注意：默认创建的新用户是没有任何的权限。
		查看新建的用户权限：show grants;
【退出登录】exit;
	3、授权（root）
		3.1 用root登陆mysql
		3.2 执行
all privileges	所以权限
on			中前面的*号用来指定数据库名，后面的*号用来指定表名。
*.*.			用来指定权限针对哪些库和表
to			表示将权限赋予某个用户
identified by	制定用户的登录密码
with grant option	这个选项表示该用户可以将自己拥有的权限授权给别人。
			grant all privileges on *.* to '用户名'@‘地址’identified by '密码'  with grant option;
		3.3 使用新用户登陆
			mysql -u 用户名 -p
			输入密码		
		3.4  查看详细的权限：
			show privileges\G
		3.5  查询权限：
			show grants;
		3.6   刷新权限：
			flush privileges  
	4、删除用户 : 
		drop user'用户名'@'IP地址';
	5、修改用户：
rename user '原来用户名'@'localhost'to '新用户名'@'IP地址'
	6、修改用户密码：
set password for 'hgzs'@'localhost'=password(新密码)
	7、移除用户权限：
		revoke update on *.* from '用户名'@ 'localhost'；
移除用户 update权限
		show grants;(没问题) ------查询权限
		(show privileges\G)(有问题)------查询权限
	8、修改权限	
		update user set host='%' where user='root'
